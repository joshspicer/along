Along support

For help with pairing, passkeys, recovery, focus sync, or account privacy, email support@along.app.

Before writing

  • Reconnect and reopen Focus to replay the durable sync cursor.
  • Use one unused recovery code if a passkey is unavailable.
  • Include the app version and commit shown in Account and devices.

Never send a recovery code, refresh token, passkey export, or device passcode to support.
